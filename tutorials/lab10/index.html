Question 3

You need to specify

  • Prior density: \(\theta \sim \beta(\alpha, \beta)\) \[ \pi(\theta) \propto \theta^{\alpha - 1}\left(1 - \theta\right)^{\beta - 1} \]
  • Likelihood: \(X \sim B(n, p)\) \[ L(\theta) = \binom{n}{0}\theta^{0}(1 - \theta)^{n} + \cdots + \binom{n}{p}\theta^{p}(1 - \theta)^{n - p} \]

  • Posterior density: \[ \pi(\theta|\mathbf{x}) \propto \pi(\theta) \times f(\mathbf{x}|\theta) \]

  • R code
